# Sejarah dari Human Interaction computer
## Melalui terminal
Sebelum ada General User Interface, manusia memberikan instruksi kepada mesin langsung menggunakan bahasa pemrograman di terminal. Hal ini membuat komputer menjadi sangat ekslusif karena hanya beberapa orang yang bisa mengoperasikan terminal saja yang bisa mengambil manfaat dari komputer

## General User Interface
Lalu, Apple dan Microsfot mempelopori dan mempopulerkan User interface dimana program komputer dipisahkan menjadi 2 bagian, frontend yang berisi visualisasi dan program yang mentranslasi visual menjadi sekumpulan perintah yang dipahami oleh komputer. 
* Kelebihan
    * User tidak perlu menguasai atau memahami pemrograman untuk bisa memanfaatkan komputer
* Kekurangan
    * Sekarang banyak orang yang mempunyai ide namun terkendala dengan keterbatasan memahami cara mengoperasikan aplikasi.
