Miniconda is a free, miniature installation of Anaconda Distribution that includes only conda, Python, the packages they both depend on, and a > > small number of other useful packages.


### Install Miniconda
to install miniconda, click link below [link](https://www.anaconda.com/docs/getting-started/miniconda/install)

### Activate & Deactivate env
```
1. Activate env
source /path/to/miniconda/bin/activate

2. Deactivate env
conda deactivate
```
